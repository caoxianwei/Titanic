# Evaluation System

该项目是我参与浙江省大学生某比赛的一个项目,目前代码里接了后端,不适合大家拿来学习,待完成比赛后我会割除后端采用mock数据,方便大家学习.

该项目主要采用了以下技术栈
+ [ng-alain](https://github.com/cipchk/ng-alain) 企业后台模板
+ [ng-zorro-antd](https://github.com/NG-ZORRO/ng-zorro-antd) 蚂蚁金服Angular版UI组件
